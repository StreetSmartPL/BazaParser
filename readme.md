# Parser bazy
Imo proste,
Pobierasz bazę z strony rządowej

Jej nazwę zmieniasz na Baza.xlsx

Wszystkie filmy wrzucasz do folderu "videos"

I odpalasz skrypt main.py

(btw, skrypt zmienia exela do jsona)

## Wymagania

- Python 3.8
- openpyxl
- dataclass
- dataclass_json
